# Tendermint
Blockchain app baesd on tendermint 

## Technologies used
1. Tendermint concensys
2. Shea
3. Lotionjs

## usuage 
1. Install dependencies 
```
$ npm i
``` 

2. Install shea to connect to blockchain
```
$ npm i -g shea
```

3. RUN the app
```
$ node app.js
```

4. connect to blockchain

```
$ shea <GCI>
```
